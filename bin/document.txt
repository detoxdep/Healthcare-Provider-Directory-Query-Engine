III. DESIGN QUALITY & PROGRAMMING STYLE

Inheritance: A base Person class encapsulates shared attributes, while Doctor and Patient extend it to handle role-specific data.

Encapsulation: All attributes are private, with data access strictly managed through public Getter/Setter methods.

Naming Conventions: Strict adherence to Java camelCase and PascalCase (e.g., calculateDistance, testSearch).

Code Scalability: Methods are kept concise (under 40 lines) to ensure the search loop remains performant.

IV. INSTRUCTIONS: HOW TO RUN THE SYSTEM

Prerequisites: JDK 11+ and the JUnit 4 library.

Compilation: Open the project in Eclipse or use the terminal: javac -d bin src/*.java test/*.java.

Execution: Run the Main.java file to launch the interactive terminal UI.

Unit Testing: In Eclipse, right-click the test package and select Run As > JUnit Test to verify the Green Bar status across all 6 test files.
